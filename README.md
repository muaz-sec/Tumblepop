# TumblePOP Game (C++ SFML)

## Description
A simple 2D platformer game made in C++ using SFML for Semester 1 project.

## Features
- Tile-based level system
- Gravity and jumping
- Collision detection
- Background music
- Keyboard controls (WASD / Arrow keys)

## Controls
- A / Left Arrow → Move left
- D / Right Arrow → Move right
- W / Space / Up → Jump
- Esc → Exit

## Build Instructions (Linux)
cmake .
make
./TumblePOP

## Dependencies
- SFML 2.5+

## 🎯 Project Purpose
This project was developed as a Semester 1 assignment to understand core game development concepts such as:
- Game loops
- Physics simulation (gravity & jumping)
- Collision detection
- Real-time input handling
- Rendering using SFML
